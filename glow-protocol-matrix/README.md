# Bestie Glow Protocol — Full Biomarker Matrix

Internal operations document mapping all Glow Panel biomarkers to lifestyle, supplement, skincare, and treatment resolutions.

## Contents

- `Bestie_Glow_Protocol_Full_Matrix.pdf` — The ready-to-use PDF
- `generate_matrix.py` — Python script to regenerate/edit the PDF

## To regenerate the PDF

```bash
pip install reportlab
python generate_matrix.py
```

## Markers covered

- TSH, Free T3, Free T4 (Thyroid)
- Vitamin D, Ferritin, B12 (Vitamins & Nutrients)
- hs-CRP, Homocysteine (Inflammation)
- Estradiol, Cortisol (Hormones)
- HbA1c, Fasting Insulin (Metabolic)
